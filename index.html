A new connectivity solution for remote data gathering

UNMANNED AERIAL SYSTEM

Our long-range fixed-wing Data Mule UAS autonomously flies to and wirelessly gathers data from remote sensors. The Data Mule UAS can service data stations up to 40km from the mission base station.

GROUND-BASED DATA STATION

Each weatherproof data station is compatible with most existing sensors including camera traps, air quality sensors, and soil monitors. As the UAS circles overhead, the data station transfers the fully encrypted sensor data.
